#CodeFOLIO란?
![CodeFOLIO?] (http://oursoccer.co.kr/study/reserve/img/codefolio_origin.PNG)

####**깃허브 + 블로그 = CodeFOLIO**
사용자의 **소스 commit**, **블로그 포스팅**, 각종 이벤트를 **타임라인**으로 구성하여
하나의 **포트폴리오**를 만들어주는 웹서비스
